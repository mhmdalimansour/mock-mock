User Management API

Get All Users

GET /api/users

Response:
{
  "users": [
    {
      "id": 1,
      "name": "John Doe",
      "email": "john@example.com"
    }
  ]
}

Create User

POST /api/users Request: { "name": "string", "email": "string" } Response: { "id": 1, "name": "string", "email": "string" } Status: 201

Update User

PUT /api/users/1 Request: { "name": "string", "email": "string" } Response: { "id": 1, "name": "string", "email": "string" }

Delete User

DELETE /api/users/1

Response:
{
  "success": true
}
